BigCSV.co is a Next.js 16 + Supabase SaaS app for BigCommerce merchants. Core tools: CSV Exporter, Product Library (with Claude AI summaries), Price Adjuster, Sales Book Builder (PDF), Time Capsule (snapshots). Dark-themed UI, AES-256 encrypted credentials, deployed at bigcsv.co.

**Why:** Built to solve BigCommerce's limited native product management — merchants need custom CSV exports, bulk pricing, and sales materials but BigCommerce doesn't provide good tools for these.

**How to apply:** This is the core product context. Use it to understand feature references, API routes, and user workflows when making changes.
